 - Open root of repository with AppCode, install plugin KMM for AppCode.

### Run
 - Choose run configuration iOS App

### Run on real iOS device
 - Device should be at least with iOS 16.5
 - Find your TEAM_ID like in this instruction https://github.com/JetBrains/compose-multiplatform-template#running-on-a-real-ios-device
 - Create file project.properties in the root of repository. Add TEAM_ID=[your team id]
